# Employee-Promotion-Dashboard

An interactive HR analytics dashboard developed using Power BI to analyze workforce performance, employee promotions, and organizational insights.

## Project Overview
This project focuses on analyzing employee data to identify promotion patterns, workforce distribution, and performance indicators through interactive visualizations and KPIs.

## Data Preparation
- Performed data cleaning and preprocessing
- Removed duplicates and handled missing values
- Created relationships between tables using Power BI Model View
- Organized employee and regional data for analysis

## Dashboard Features
- Employee distribution by department
- Promotion insights and analysis
- Employee education level breakdown
- Age group analysis
- City and regional employee distribution
- Average training score KPI
- Employee service length analysis

## Tools & Technologies
- Power BI
- Power Query
- Data Cleaning
- Data Modeling
- Microsoft Excel

## Key KPIs
- Total Employees: 55K
- Promoted Employees: 4668
- Average Training Score: 63.39
- Maximum Length of Service: 37 Years
